# Mintlify Documentation Site Clone (Desktop)

A **desktop-only HTML & CSS recreation of a documentation-style website inspired by Mintlify**, focused on **content structure, readability, sidebar layout, and visual accuracy**.

The objective was to closely match the **documentation layout and hierarchy** of the original site, without adding interactivity.

---

## Sections Recreated

- Top Navigation Bar
- Hero Section with headline, description, email input & CTA
- Documentation Preview (static sidebar + content cards)
- Trusted By / Logos
- Feature Highlights
- Intelligent Assistant / UI Preview
- Enterprise Features
- Case Studies / Customer Stories
- Final Call-To-Action
- Footer

---

## Tech & Constraints

- HTML5, CSS3
- No JavaScript
- No TailwindCSS
- No animations
- Desktop-only (not responsive)
